## Rodando localmente

```bash
yarn install
yarn dev
```

Acesse em: `http://localhost:5173`

## Tech Stack deste portfólio

- **React 18** + **TypeScript**
- **Vite** como bundler
- **Tailwind CSS** para estilização
- **React Router DOM v7** para navegação
- **GitHub REST API** para dados em tempo real
